Программа состоит из 3 меню(Home, Tools, Settings),которые в свою очередь состоят из следующих компонентов Menu 1.1 В разделе Area мы можем добавить, редактировать, и удалять (Region ,City Address,service и comments) 1.2 В разделе Filter and Search мы можем отфильтровать список и осуществить поиск. 1.3 В разделе Technics мы можем добавить, редактировать, и удалять информацию которая связанна с техникой (в нашем примере это компьютеры) 1.4 В разделе Scan and Remote мы можем просканировать проблемы с компьютерами которые находятся в заданном диапазоне, нажав на кнопку Remote мы можем подключится к этой системе. 1.5 В разделе Task работник получает свои таски. 1.6 В разделе Export мы экспортируем наши данные в excel файл. 1.7 Тут есть два поля 1 Города 2 компьютеры В конце StatusBar в котором показано число городов, компьютеров, заданной стоимости, количество Task-ов, и количество просканированных компьютеров, которое указывает количество активных и пассивных компьютеров.

Tools 2.1 В разделе с помощью Ip scan сканируем свободные и несвободные ip. 2.2 С помощью Ping проверяем подключен ли компьютер к сети, если да, та же команда проверяет скорость интернета. 2.3 С помощью Graph показывает активное состояние интернета с помощью графика. 2.4 С помощью Explorer можем обменят файлы с компютерами.

Settings. 3.1 С помощью Rdp-yastex контролируем дистанционное подключение. 3.2 С помощью users добовляем, редактируем, удаляем, и доем доступы user-ом. 3.3 С помощью Database делаем backup и restore данных
